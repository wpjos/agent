# tsas-num-prep-cleaner 算法规格

> 本文档定义 cleaner 的全部处理规则和算法逻辑，供 SKILL.md 引用。

---

## 1. 处理流程

### 步骤 1：读取元信息

解析元信息 MD，建立每列的角色、数据类型、单位等信息。元信息格式见 `metadata_spec.md`。

### 步骤 2：显式缺失处理

遍历所有非时间列单元格，识别 NA 标记字符串：
```
NA_TOKENS = ["", "NA", "N/A", "#N/A", "NaN", "null", "None", "-", "--", "nil"]
```
- 大小写不敏感，单独成值时识别
- 统一替换为空字符串 `""`（保持 dtype=str）
- 记录到报告

### 步骤 3：类型归一

**仅对业务数据列执行**（标注列、类别列跳过）。

#### 判定逻辑：
1. 剔除 NA 后，统计每个单元格类型（int / float / complex / str）
2. 计算各类型占比
3. 决策：

| 场景 | 规则 |
|---|---|
| 主导类型占比 ≥ 90% | 少量高格数据**降级**到主导类型；降级不成功置空值 |
| 高格数据占比 > 10% | **升格**全部到最高类型（int→float→complex→str） |

#### 类型层级（从低到高）：
```
int < float < complex < str
```

#### 降级规则：
- int 目标：`try_parse_int()` 成功则取整数值
- float 目标：`try_parse_float()` 成功则取浮点值
- complex 目标：`try_parse_complex()` 成功则取复数值
- 降级失败 → 置空值

#### 升格规则：
- 对每个非主导类型单元格，尝试转换到最高类型
- 转换失败 → 置空值

**所有修改记录到报告。**

### 步骤 4：单位归一

**仅对有单位信息的业务数据列执行。**

#### 目标单位确定：
- 用户指定 `target_units`（来自元信息或 CLI）→ 遵循
- 用户未指定 → **不执行转换**（保留原始单位，仅记录到报告）

#### 换算规则：
- 同物理量类别的单位可互相转换（如 mA → A）
- 不同物理量类别不可转换（如 A → V）→ 报错
- 温度特殊处理：℃ / ℉ / K 之间非线性换算
- 未知单位 → 警告，不做转换

### 步骤 5：类别编码

**仅对类别列执行。**

- 用户指定映射（`category_maps`）→ 遵循
- 用户未指定 → **首次出现顺序** → 整数（0, 1, 2, ...）
- 输出为整数值
- 编码映射表记录到报告

### 步骤 6：标注编码

**仅对标注列执行。**

- 检测当前编码方式：
  - 取值 ∈ {0, 1} → 已是目标格式，无需转换
  - 取值 ∈ {-1, 1} → 转换：`1→0`（正常），`-1→1`（异常）
  - 其他取值 → **报错**

### 步骤 7：异常值/哨兵值（可选，默认关闭）

仅当用户传入 `--enable-outlier-detection` 时执行：
- 有效值域检查（`value_range`）：超出 `[min, max]` 的值标记为空
- 哨兵值检查（`sentinels`）：匹配哨兵值列表的值标记为空
- 检出记录到报告

### 步骤 8：输出

- 治理后数据写入 `cleaned_data.csv`
- 治理报告写入 `cleaned_data_report.md`

## 3. 关键算法细节

### 3.1 类型层级与归一

```
int (最低) → float → complex → str (最高)
```

主导类型占比 = 主导类型单元格数 / 非空单元格总数

### 3.2 单位换算

线性比例换算公式：
```
新值 = 原值 × (源单位因子 / 目标单位因子)
```

温度换算：
```
℃ → ℉: F = C × 9/5 + 32
℃ → K: K = C + 273.15
℉ → ℃: C = (F - 32) × 5/9
K → ℃: C = K - 273.15
```

已知单位类别：
- current（电流）：a, ma, ua
- voltage（电压）：v, mv, kv
- power（功率）：w, kw, mw
- temperature（温度）：celsius, fahrenheit, k
- frequency（频率）：hz, khz, mhz
- pressure（压力）：pa, kpa, mpa, bar, mbar
- mass（质量）：g, mg, kg, t
- length（长度）：mm, cm, m, km, um
- duration（时间间隔）：s, ms, us

### 3.3 类别编码

```python
code_map = OrderedDict()
next_code = 0
for value in column:
    if value not in code_map:
        code_map[value] = next_code
        next_code += 1
    output.append(code_map[value])
```

### 3.4 标注编码判定

```
if 取值集合 ⊆ {0, 1}:     → 无需转换
elif 取值集合 ⊆ {-1, 1}:   → 转换 (1→0, -1→1)
else:                      → 报错
```
