## 示例 2：多目标优化（最大化产率 + 最小化成本）

### Scenario 1：初始化

```bash
python main.py --mode init \
  --description "药物合成工艺优化：反应温度60-120°C，反应时间1-8h，溶剂选择（乙醇/丙酮/甲醇），催化剂浓度0.01-0.5mol/L。目标：最大化产率，同时最小化原料成本（成本与催化剂浓度和反应时间正相关）"
```

**输出**（建议书片段）：

```
### 参数空间

| 参数名 | 类型 | 下界 | 上界 / 类别 | 说明 |
|--------|------|------|-------------|------|
| temperature | num | 60 | 120 | 反应温度 (°C) |
| reaction_time | num | 1 | 8 | 反应时间 (h) |
| solvent | cat | — | ['乙醇', '丙酮', '甲醇'] | 溶剂类型 |
| catalyst_conc | pow | 0.01 | 0.5 | 催化剂浓度 (mol/L)，base=10 |

### 优化目标

| 目标名 | 优化方向 | 说明 |
|--------|---------|------|
| yield | 最大化 (max) | 产率 (%) |
| cost | 最小化 (min) | 原料成本 (元) |
```

### Scenario 2：多轮迭代后的多目标推荐


## 示例 3：含类别参数 + 布尔参数

```bash
python main.py --mode init \
  --description "材料合成：烧结温度800-1200°C，保温时间1-6h，气氛选择（空气/氮气/氩气），是否加压（True/False），目标最小化晶粒尺寸"
```
